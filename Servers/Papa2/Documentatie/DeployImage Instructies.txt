In SCCM Task Sequence Maken
Stappen:

1:  Software Library -> Operating Systems -> Task Seaquences (rechtermuisklik) -> Create Task Sequence
2:  SELECTEER "Create a new Custom Task Sequence" -> Next
3:  SELECTEER de "Windows10 x64" boot image -> Next
4: Task Sequence AANGEMAAKT
5: Applicaties invoeren: "Application Management" -> RECHTERMUIS "APPLICATIONS"
6: SELECTEER "Create Application" -> IMPORTEER VANAF SERVER (\\DC1\sharedfolder\setup.msi)
7: Vul velden in, CONTROLEER de COMMAND LINE voor de INSTALLATIE
8: SELECTEER bij "Install Behavior": "INSTALL FOR SYSTEM" -> NEXT -> NEXT
