APPLICATIES TOEVOEGEN AAN SCCM
PREREQ: 
    1: RIGHT-CLICK "Applications" -> "Create Application"
    2: Browse naar UNC Pad, bv: "\\DC3\Applicaties\AcroRead.msi" 
    3: NEXT -> NEXT
    4: BIJ "Installation Program": voeg "/norestart" toe op EINDE
    5: BIJ "Install Behavior": "Install for System"
    6: NEXT -> NEXT -> CLOSE
    !!!     HERHAAL VOOR ELKE APPLICATIE    !!!

APPLICATIES TOEVOEGEN AAN TASK SEQUENCE
1: RIGHT-CLICK de gemaakte TASK SEQUENCE -> "Edit"
2: BIJ "STATE RESTORE / INSTALL APPLICATIONS"
    - SELECTEER "Install Application"
    - SELECTEER "Install the following Applications"
    - KLIK op STER
    - VINK de 4 Applicaties AAN -> OK
    - CONTROLEER of ONDERSTE PROPERTY is AANGEVINKT (IF 1 FAILS, CONTINUE WITH OTHER)
    - APPLY -> OK
